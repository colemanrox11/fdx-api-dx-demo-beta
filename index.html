What is FDX?

FDX provides standard for safe, secure, interoperable data exchange between Data Providers and Third Parties, specifically through the following specifications:

  • API Data Format => hyperlink directly here
  • User Experience => hyperlink directly here
  • API Security => hyperlink directly here

How can FDX be used?

FDX provides data formatting standards for several different types of data. These are organized by product, each with their own set of API endpoints, API models / structures, and accompanying reference documentation in the user experience and security guidelines.

FDX API Products: Consent

Overview

The FDX Consent API defines how Data Providers and Third Parties manage consumer-permissioned data-sharing agreements in a standardized way. Within the FDX Standards, it establishes a consistent lifecycle model for consent - covering creation, retrieval, update, and revocation - while recording key metadata such as who granted consent, what data clusters or endpoints are included, the duration of the agreement, and its current status.
