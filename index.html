How can FDX be used?

With FDX, you can access or provide access to standardized financial data via our API format for common use cases:

  • Account Information – Account types, ownership, and status
  • Balances – Real-time or historical account balances
  • Transactions – Categorized transaction histories across account types
  • Customer Profiles – End-user information and associated relationships
  • Statements – Monthly or periodic account statements
  • Investments – Positions, holdings, and performance metrics
  • Loans & Liabilities – Credit cards, mortgages, student loans, and more
  • Tax - Retrieve, create, or update tax forms for customers.
  • Payroll - Retrieve employee payroll reports.
  • Money Movement - Retrieve and create payment, payee, payment initiation, and account-to-account transfer data

FDX supports both Data Recipients and Data Providers, using secure tokenized authentication flows and a shared data model defined by working groups focused on API, Security, and UX standards.
